# 🚀 Skilla – Your AI Interview Simulator

## 💡 Inspiration  
Interviews can be intimidating, especially without proper practice or feedback. Many students and job seekers don’t have access to real interview environments where they can build confidence and improve their answers.

That’s why I built **Skilla** — an AI-powered interview simulator that helps users practice smarter, gain confidence, and improve their communication skills in a realistic way.

---
## 🌐Live URL: **https://skilla-ai.streamlit.app**


## 🤖 What It Does  
**Skilla** is a smart AI interview coach that:
- 🎤 Simulates real interview scenarios  
- 🧠 Asks role-specific and adaptive questions  
- 💬 Provides instant feedback and improvements  
- 📊 Helps users refine answers and communication  
- 🎯 Acts like a personal mentor for interview preparation  

It’s designed to feel like a **real interviewer**, not just a chatbot.

---

## ⚡ How I Built It (Vibe Coding 🔥)  

Skilla was created using a **vibe coding approach** — rapidly building and iterating using AI tools.

### 🧠 Tools Used:
- **Gemini** → for generating UI, structure, and frontend code  
- **Groq API** → for ultra-fast AI responses  
- **Streamlit** → single-file lightweight app  

### 🛠️ Process:
1. I started with a simple idea and gave prompts to **Gemini** to generate a **clean, modern UI**.  
2. Then I enhanced the prompts to make the chatbot behave like a **professional interviewer**.  
3. Integrated **Groq API** to handle real-time AI responses with low latency.  
4. Iterated multiple times by tweaking prompts (this is where the *vibe coding magic* happens ✨).  
5. Kept everything simple by building it as a **single HTML file** for easy deployment.  

> 💬 No complex setup. Just ideas + prompts + iteration.
